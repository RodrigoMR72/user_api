## 1. Inicializando o projeto

-y: seta todas as respostas como YES

```js
npm init -y
```

## 2. Instalando o o nodemon e o express

-y: seta todas as respostas como YES

```js
npm i nodemon express
```


Configurar o _package_ para que nosso servidor execute com npm start
